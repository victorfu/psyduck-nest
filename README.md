# Psyduck-Nest

Psyduck-Nest is a boilerplate for NestJS with TypeORM, and JWT Authentication. It is a simple and easy to use boilerplate for starting a new project with NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
